# Lesson 6: TeamPact Assessments

## Goal

Practice using a TeamPact import function and exploring assessment results.

## Dataset

Use `import_teampact_assessment_data()`.

Choose one survey ID from `AVAILABLE_TEAMPACT_ASSESSMENT_SURVEYS`.

## Your Tasks

1. Load one TeamPact assessment survey into a dataframe.
2. Show the dataframe in Streamlit.
3. Show the list of columns.
4. Count how many learners are in each grade.
5. Count how many learners are in each class.
6. Sort the dataframe by response date.

## Questions To Answer

- How many responses were collected?
- Which grades appear in this dataset?
- Which classes have the most responses?

## Stretch Task

Choose one score column and calculate the average.

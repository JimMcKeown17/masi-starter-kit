# Lesson 3: Group And Sort

## Goal

Practice grouping data and sorting summary tables.

## Dataset

Use `import_2025_childrens_results()`.

## Your Tasks

1. Load the 2025 children dataset.
2. Group the data by school.
3. Count how many learners are in each school.
4. Sort the results from highest to lowest.
5. Show the top 10 schools.
6. Repeat the same task for mentor or class.

## Questions To Answer

- Which school has the most learners?
- Which mentor appears most often?
- Which class appears most often?

## Stretch Task

Create a simple bar chart in Streamlit for the top 10 schools.

# Lessons

These lessons are written for beginners.

The goal is to help staff:

- import a real Masi dataset
- store it in a dataframe
- inspect the dataframe
- answer simple questions with pandas
- show results in Streamlit

Important:

- There is no starter code in these lessons.
- You should write the code yourself in `main.py`.
- Work through one lesson at a time.

Suggested order:

1. `01_first_local_dataset.md`
2. `02_filter_and_count.md`
3. `03_group_and_sort.md`
4. `04_airtable_schools.md`
5. `05_airtable_literacy_sessions.md`
6. `06_teampact_assessments.md`
7. `07_teampact_sessions.md`

General workflow for every lesson:

1. Choose one import function
2. Load the data into a dataframe
3. Show the dataframe in Streamlit
4. Use pandas to answer a few questions
5. Display the answers clearly

# Lesson 4: Airtable Schools

## Goal

Practice importing a live Airtable dataset and exploring the columns.

## Dataset

Use `import_airtable_schools()`.

## Your Tasks

1. Load the Airtable schools dataset.
2. Show the dataframe in Streamlit.
3. Show the first 10 rows.
4. Show the columns.
5. Count how many schools are in each type.
6. Sort the schools alphabetically by school name.

## Questions To Answer

- How many schools are in the dataset?
- Which school types are included?
- Which columns look like location columns?

## Stretch Task

Display only the columns that are easiest for a beginner to understand.

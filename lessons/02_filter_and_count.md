# Lesson 2: Filter And Count

## Goal

Practice selecting rows and counting values.

## Dataset

Use `import_2025_childrens_results()`.

## Your Tasks

1. Load the 2025 children dataset into a dataframe.
2. Show only the rows for one language.
3. Show only the rows for one grade.
4. Count how many learners are in each grade.
5. Count how many learners are in each language.
6. Show the results in Streamlit.

## Questions To Answer

- Which grade has the most learners?
- Which language has the most learners?
- How many learners are in the grade you selected?

## Stretch Task

Filter to one school and one language at the same time.

# Lesson 1: First Local Dataset

## Goal

Load one local Masi dataset and get comfortable with a dataframe.

## Dataset

Use `import_2024_childrens_results()`.

## Your Tasks

1. Import the function into `main.py`.
2. Run the function and save the result in a dataframe variable.
3. Show the full dataframe in Streamlit.
4. Show the first 5 rows.
5. Show the list of columns.
6. Show the number of rows.
7. Show the number of columns.

## Questions To Answer

- What does one row represent?
- Which columns look most useful?
- Which columns look like categories?
- Which columns look numeric?

## Stretch Task

Choose 3 columns and explain in plain English what each one means.

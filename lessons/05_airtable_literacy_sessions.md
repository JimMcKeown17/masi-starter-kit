# Lesson 5: Airtable Literacy Sessions

## Goal

Practice working with a live sessions dataset.

## Dataset

Use `import_airtable_literacy_sessions()`.

## Your Tasks

1. Load the literacy sessions dataset.
2. Show the dataframe.
3. Show the first 10 rows.
4. Sort the dataframe by session date.
5. Count how many session rows there are.
6. Count how many sessions there are for each school.
7. Count how many sessions there are for each literacy coach.

## Questions To Answer

- Which school has the most sessions?
- Which literacy coach has the most sessions?
- What does one row represent?

## Stretch Task

Filter to one school and look only at that school’s sessions.

# Lesson 7: TeamPact Sessions

## Goal

Practice working with TeamPact session data.

## Dataset

Use `import_teampact_sessions_data()`.

## Your Tasks

1. Load the TeamPact sessions dataframe.
2. Show the dataframe in Streamlit.
3. Show the first 10 rows.
4. Count how many sessions belong to each program.
5. Count how many sessions belong to each class.
6. Sort the dataframe by session date.

## Questions To Answer

- How many session rows are in the dataset?
- Which program appears most often?
- Which class appears most often?

## Stretch Task

Create a chart that shows sessions by program.
